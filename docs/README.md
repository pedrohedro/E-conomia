# E-conomia - Gestão Inteligente para E-commerce

**E-conomia** é um CRM (Customer Relationship Management) e ERP (Enterprise Resource Planning) tático, desenhado especificamente para **Donos de E-commerce** que buscam profissionalizar sua operação e ter controle total sobre seus números.

O projeto consolida dados de múltiplos marketplaces em uma interface moderna, rápida e intuitiva, permitindo uma gestão baseada em dados reais (Data-Driven).

## 🚀 Proposta de Valor

Diga adeus às planilhas complexas e à fragmentação de dados. Com o E-conomia, você centraliza:
- **Marketplaces:** Mercado Livre, Amazon, Shopee e Nuvemshop.
- **Estoque:** Controle multicanal unificado.
- **Financeiro:** Lucratividade real, impostos e pró-labore.
- **Decisão:** Insights em tempo real sobre a saúde do seu negócio.

## 🌐 Hospedagem e Deploy

O projeto está configurado para deploy automático via **Vercel**. 
- **URL de Produção:** Integrada ao repositório GitHub para que cada `git push` atualize o site instantaneamente.
- **Configuração:** O arquivo `.vercel/project.json` já contém os identificadores do projeto (`e-conomia-crm`), garantindo que o ambiente de produção esteja sempre sincronizado com o código mais recente.

## 🛠️ Funcionalidades Principais

### 📊 Dashboard Estratégico
Visão macro de faturamento, taxas e reputação. O **Health Score** indica instantaneamente se sua operação está saudável ou se precisa de atenção urgente.

### 🔌 Integrações Nativas
Conecte suas lojas através de protocolos seguros (OAuth 2.0).
- **Mercado Livre:** Sincronização de vendas e gestão de Full.
- **Amazon & Shopee:** Controle de margens e logística (FBA/DBA).
- **Nuvemshop:** Integração para e-commerce próprio.

### 📦 Inventário Inteligente
Gestão de SKU com alertas visuais de **Reposição Urgente**. Saiba exatamente quanto capital você tem imobilizado em produtos e qual a margem de cada item.

### 💸 Contábil Pro (CFO View)
Um módulo financeiro de nível executivo:
- **Fluxo de Caixa:** Calendário financeiro com previsão de entradas e saídas.
- **DRE Simplificado:** Receita Bruta vs. Líquida, Impostos e CMV.
- **Gestão de Despesas:** Registro fácil de custos fixos e variáveis.

## 💻 Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 moderno (Glassmorphism).
- **Estilização:** Tailwind CSS (Utilitário-first).
- **Ícones:** Lucide Icons.
- **Gráficos:** Chart.js para visualização de performance.

## ⚡ Como Executar
O projeto é atualmente uma aplicação frontend estática de alta performance. Para visualizar:

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Edge, Safari).
3. Navegue entre as abas laterais para explorar os diferentes módulos de gestão.
